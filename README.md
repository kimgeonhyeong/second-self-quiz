# 8초의 나 · 판단 게임

설문 아님. **5번의 결정을 8초 안에** 탁탁 내리면, 당신이 어떻게 판단하는 사람인지(사고 커널 초안)가 나오는 정적 웹 게임.

- **라이브**: https://kimgeonhyeong.github.io/second-self-quiz/
- 외부 의존성·서버·빌드 없음. `index.html` 하나로 동작.

## 어떻게 작동하나

1. **5개 딜레마**를 건당 **8초 제한**으로 결정 — 한 번 고르면 잠금(수정 불가). 머리로 계산하기 전 본능을 잡는다.
2. 고른 뒤 **"왜"를 칩 한 번 더 탭**(또는 직접 쓰기) — 선택만이 아니라 판단의 이유까지 담는다.
3. **마지막 관문**: 시스템 요약을 보기 전에, 본인이 먼저 자기 이유들을 관통하는 한 줄을 뽑는다(바넘 효과 차단·본인 확인).
4. **결과**: 별명 18종(속도 × 첫반응 × 리스크) + 평균 결정 속도 + 판단 방식 요약. 속도 타입별 등장 연출(번개/링/오로라).

결과는 커널 스키마 JSON **초안**(미확인) — 공유·복사 가능하고, 동의 시 익명 저장된다.
